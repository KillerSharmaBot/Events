# FF Events

Welcome to **FF Events** — a web app that lists live, upcoming, and past events in a clean, organized view.

This project helps users track events in one place and supports features like favorites, auto-refresh, and export options.

---

## 🔍 Features

✔️ View all events in one dashboard  
✔️ Filter by **Live**, **Upcoming**, and **Ended**  
✔️ Mark events as **Favorites**  
✔️ Auto-refresh events list  
✔️ Export events as **JSON**  
✔️ Sort by **Newest** / **Oldest**

## 🚀 Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Edge, Safari)
- Internet connection

---

## 💡 How to Use

1. Open the site in a browser:
   https://killersharmabot.github.io/Events

2. Wait for the events to load.

3. Use the filters to:
   - See **Live** events
   - See **Upcoming** events
   - See **Ended** events

4. Favorite events to save them locally.

5. Use **Export JSON** to download the current list.
